# flutter_hotel

## 项目简介

酒店管理系统练手项目APP前端，用flutter模拟美图酒店UI，后端使用springcloud微服务架构。

## github地址
后端入口: [酒店管理系统](https://github.com/lshaoshuai/hms-springcloud)

## flutter相关技术
